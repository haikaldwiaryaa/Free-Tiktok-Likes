# 🚀 TikTok Booster

A simple web tool to send free likes to TikTok videos instantly. Built with a space/astronaut themed UI.

## ✨ Features

- 🧑‍🚀 **Space-Themed UI** — Deep sky background with twinkling stars & glassmorphism design
- ⚡ **Super Fast** — Instant like delivery
- 🛡️ **Safe** — No login or password required
- 📱 **Responsive** — Works on mobile & desktop
- 🌠 **Smooth Animations** — Floating astronaut, shooting stars, fade-in effects

## 🛠️ Installation

### Deploy on Vercel (Recommended)

1. **Fork** this repository
2. Go to [vercel.com](https://vercel.com) and sign in with GitHub
3. Click **"New Project"** → Import your forked repo
4. Click **Deploy** — done! 🎉

### Run Locally

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/tiktok-like.git

# Go to the project folder
cd tiktok-like

# Install dependencies
npm install

# Start the server (requires Vercel CLI)
npx vercel dev
```

Then open `http://localhost:3000` in your browser.

## 📝 How to Use

1. Open TikTok app & copy your video link
2. Paste the link in the input box
3. Click **"Kirim Likes Gratis"** and wait
